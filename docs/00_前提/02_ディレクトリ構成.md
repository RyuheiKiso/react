# バックエンド

VerticalSliceArchitectureで構成したい

``` 
/server
├─ Features                 # 各機能ごとのディレクトリ
│  ├─ Login                 # ログイン機能
│  │  ├─ Controllers        # コントローラー
│  │  │  └─ LoginController.cs
│  │  ├─ Services           # ビジネスロジック
│  │  │  └─ LoginService.cs
│  │  ├─ Models             # データモデル
│  │  │  └─ LoginModel.cs
│  │  ├─ Repositories       # データアクセス層
│  │  │  └─ LoginRepository.cs
│  │  └─ DTOs               # データ転送オブジェクト
│  │     └─ LoginDto.cs
│  └─ Search                # database検索機能
│     ├─ Controllers        # コントローラー
│     │  └─ SearchController.cs
│     ├─ Services           # ビジネスロジック
│     │  └─ SearchService.cs
│     ├─ Models             # データモデル
│     │  └─ SearchModel.cs
│     ├─ Repositories       # データアクセス層
│     │  └─ SearchRepository.cs
│     └─ DTOs               # データ転送オブジェクト
│        └─ SearchDto.cs
├─ Common                   # 共通機能
│  ├─ Middleware            # ミドルウェア
│  │  └─ CustomMiddleware.cs
│  ├─ Utilities             # ユーティリティ
│  │  └─ UtilityHelper.cs
│  ├─ Extensions            # 拡張メソッド
│  │  └─ StringExtensions.cs
│  ├─ Configurations        # 設定関連
│  │  └─ AppConfiguration.cs
│  ├─ API                   # REST API, gRPC, QUIC共通機能
│  │  ├─ Rest               # REST API関連
│  │  │  └─ RestHandler.cs
│  │  ├─ Grpc               # gRPC関連
│  │  │  └─ GrpcHandler.cs
│  │  └─ Quic               # QUIC関連
│  │     └─ QuicHandler.cs
│  ├─ Queries               # SQLクエリの一元管理
│  │  └─ SqlQueries.cs      # クエリ定義クラス
│  └─ Logging               # ログ出力
│     └─ Logger.cs          # ログ出力ユーティリティ
├─ Config                   # 設定ファイル
│  └─ database.yaml         # データベース設定
├─ Infrastructure           # インフラ層
│  ├─ Database              # データベース関連
│  │  ├─ SqlServerContext.cs
│  │  ├─ PostgresContext.cs
│  └─ WebSocket             # WebSocket関連
│     ├─ WebSocketHandler.cs
│     ├─ WebSocketManager.cs
│     └─ WebSocketMessage.cs
├─ Tests                    # ユニットテスト
│  ├─ Features              # 各機能のテスト
│  │  ├─ LoginTests.cs      # ログイン機能のテスト
│  │  └─ SearchTests.cs     # 検索機能のテスト
│  ├─ CommonTests           # 共通機能のテスト
│  │  ├─ MiddlewareTests.cs # ミドルウェアのテスト
│  │  └─ UtilityTests.cs    # ユーティリティのテスト
│  └─ InfrastructureTests   # インフラ層のテスト
│     ├─ DatabaseTests.cs   # データベース関連のテスト
│     └─ WebSocketTests.cs  # WebSocket関連のテスト
├─ Program.cs               # エントリーポイント
├─ appsettings.json         # アプリケーション設定
├─ appsettings.Development.json # 開発用設定
└─ sample1.csproj           # プロジェクトファイル
```
